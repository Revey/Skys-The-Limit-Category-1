# Team Logos

Place team logo PNG files in this folder. The app expects the following filenames:

|----------|------|
| `cloud9.png` | Cloud9 |
| `nrg.png` | NRG |
| `furia.png` | FURIA |
| `g2.png` | G2 Esports |
| `mibr.png` | MIBR |
| `kru.png` | KRÜ Esports |
| `loud.png` | LOUD |
| `100t.png` | 100 Thieves |
| `leviatan.png` | Leviatán Esports |
| `prx.png` | Paper Rex |

| `2game.png` | 2GAME eSports |
| `sentinels.png` | Sentinels |

## Download Sources

You can download official team logos from:
- [Liquipedia Valorant](https://liquipedia.net/valorant/)
- [VLR.gg](https://www.vlr.gg/)

## Recommended Size

- 200x200 pixels or larger
- PNG format with transparency
- Square aspect ratio preferred

## Fallback Behavior

If a logo file is missing, the app will display the team's initial letter(s) instead.
